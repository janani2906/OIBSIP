# House Price Prediction

## Objective
Build a Machine Learning model to predict house prices using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Features
- Data Loading
- Data Preprocessing
- Data Visualization
- Train-Test Split
- Linear Regression Model
- Model Evaluation (R² Score, MAE, MSE)
- Price Prediction

## Dataset
Housing Dataset
