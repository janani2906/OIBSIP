## Objective
The objective of this project is to perform Exploratory Data Analysis (EDA) on a retail sales dataset to identify sales trends, customer demographics, product performance, and business insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Dataset Inspection
- Missing Value Analysis
- Descriptive Statistics
- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Demographics Analysis
- Revenue by Product Category
- Correlation Heatmap
- Additional Visualization
- Business Recommendations
