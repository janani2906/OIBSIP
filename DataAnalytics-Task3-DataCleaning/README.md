# Data Cleaning using Housing Dataset

## Objective
This project demonstrates professional data cleaning using Python, Pandas, and NumPy.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Dataset
Housing Dataset

## Tasks Performed
- Loaded the dataset
- Checked missing values
- Removed duplicate rows
- Standardized data
- Detected and handled outliers using the IQR method
- Verified data types
- Saved the cleaned dataset

## Files
- Housing_Data_Cleaning.ipynb
- housing_cleaned.csv
